This is an example post.

You can write `python` code:

```python
def hello():
    print("Hello, World!")
```

You can write `bash` code:

```bash
echo "Hello, World!"
```

You can write `sql` code:

```sql
SELECT * FROM table
WHERE column = 'value';
```

Math equations are not supported by default.
